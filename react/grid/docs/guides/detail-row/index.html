React Grid Detail Row

The “Detail Row” feature displays data row details on an expandable pane. The pane can be expanded and collapsed either programmatically or via the Grid’s UI.

Related Plugins

The “Detail Row” feature requires the following plugins:

  • RowDetailState - controls the detail rows’ expanded status
  • TableRowDetail - renders detail rows

Note that the plugin order is important.

Detail Row Setup

Add the required plugins listed above to the Grid and specify the detail row template via the TableRowDetail plugin’s template property to set up a simple Grid with detail rows.

Uncontrolled Mode

In the uncontrolled mode, you can specify the initially expanded detail rows using the RowDetailState plugin’s defaultExpandedRows property.

Controlled Mode

In the controlled mode, pass an array of the expanded row IDs to the RowDetailState plugin’s expandedRows property and handle the onExpandedRowsChange event to control the detail rows’ expanded state externally.
